# Cara Setup DARVIS di GPT Builder

## Langkah-langkah:

### 1. Buka GPT Builder
- Buka https://chatgpt.com/gpts/editor
- Klik "Create a GPT"

### 2. Tab "Configure"

**Name:** DARVIS

**Description:** AI thinking framework distributor — partner berpikir strategis, reflektif, dan multi-perspektif.

**Instructions:** Copy-paste SELURUH isi file `DARVIS_GPTs_INSTRUCTIONS.md` ke kolom Instructions.

**Conversation starters (opsional):**
- Gw lagi bimbang soal keputusan bisnis nih
- Challenge gw dong, gw mau ambil langkah ini
- Bedah dari berbagai sudut pandang
- Gw butuh sparring partner buat mikirin ini

### 3. Upload Knowledge Files
Upload semua file berikut sebagai Knowledge:
- `DARVIS_PROFILE_DR.md` (WAJIB — ini profil DR)
- `DARVIS_NODE_SolidGroup.md`
- `DARVIS_NODE_BIAS.md`
- `DARVIS_NODE_AiSG.md`
- `DARVIS_NODE_NM.md`
- `DARVIS_NODE_RISK_GUARD.md`
- `DARVIS_NODE_COMPLIANCE.md`
- `DARVIS_NODE_RESOURCES.md`

### 4. Capabilities
- Centang: Web Browsing (opsional)
- Centang: Code Interpreter (opsional)
- DALL-E: terserah

### 5. Save
- Pilih "Only me" kalau cuma untuk pribadi
- Klik Save/Publish

---

## Catatan Penting

### Yang BISA di GPTs:
- Semua gaya jawaban DARVIS (tek-tok, santai, challenge, multi-perspektif)
- Adaptive tone berdasarkan sapaan
- Deteksi emosi
- Anti echo-chamber
- Strategic challenger
- Context mode engine
- Semua knowledge dari node files
- Voice mode (pakai Advanced Voice bawaan ChatGPT)

### Yang TIDAK BISA di GPTs:
- Secretary system (meeting, action items, projects) — butuh database
- Auto-learn preferences — GPTs tidak punya persistent storage antar sesi
- Profile enrichment otomatis — tidak ada database
- Multi-mode (Mirror/Twin/Contributor) — tidak ada login system
- Notifikasi & reminder — tidak ada backend
- People database & alias resolution — tidak ada database

GPTs ini = DARVIS "lite" — obrolan dan framework thinking-nya sama, tapi tanpa fitur yang butuh backend/database.
